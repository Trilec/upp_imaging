# pystring

Stable user-facing pystring include path.

Use this package from normal application code. It delegates to `pystring_src`
for the compiled implementation.

## Include path

```cpp
#include <pystring/pystring.h>
```

## Consumer

- normal application code

## Dependency

- `pystring_src`

## Validated scope

- split
- join
- strip
- lower / upper
- startswith / endswith
- replace
- find
